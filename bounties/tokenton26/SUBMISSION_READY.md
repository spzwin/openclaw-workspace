## 📝 项目描述（长版，200-300字）

```
SolMind is an autonomous AI investment agent built specifically for the Solana ecosystem.

Core Features:
• Real-time on-chain monitoring: Tracks SOL, USDC, JUP, and BONK prices via Jupiter 
  Price API v4, plus live Solana network stats (TPS, block height, total supply)
• AI-powered analysis: Feeds structured blockchain data to GPT-4o-mini, producing 
  market summaries, per-token signals (bullish/bearish/neutral), risk assessments, 
  and actionable recommendations
• Autonomous agent loop: Runs on-demand analysis cycles, maintaining a rolling history 
  of up to 20 reports for trend detection
• Token-gated access: SPL token balance verification gates premium AI features, 
  demonstrating real token utility on Solana

Tech Stack:
- Frontend: Next.js 14 + TypeScript + Tailwind CSS
- Blockchain: @solana/web3.js (Solana mainnet-beta RPC)
- Price data: Jupiter Price API v4 (free, no key required)
- AI Engine: OpenAI GPT-4o-mini
- Deployment: Vercel (production-ready)

SolMind addresses a real gap: most Solana users lack the tools to process on-chain 
signals systematically. By combining live blockchain data with AI reasoning, SolMind 
delivers institutional-grade analysis to retail users.
```
